시장 알림 코드 운영 가이드

이 페이지는 market_regime_score.py 기준으로 시장 종합 알림 + 테마 RSI 과매도 알림 동작을 한눈에 확인하기 위한 안내 문서입니다.

최근 정리: -

시스템 개요

알림 1: 시장 종합

  • 금리차(10Y-2Y), VIX, DXY, WTI, MOVE, NAS100 자동조회
  • 공격/방어 조건 + 점수 + 패턴으로 스탠스 결정
  • 점수/패턴 가이드 문구 포함

알림 2: 테마 RSI

  • 우주/양자/원자력 각 50종목 (총 150종목)
  • 일봉 RSI14, 주봉 RSI14 과매도 탐지
  • 조건: 일봉 ≤ 30 또는 주봉 ≤ 30

스케줄

  • 매일 오전 8:50 (KST) 자동 실행
  • 텔레그램 2건 연속 전송
  • 실패 시 에러 원인 확인 가능

판정 로직

핵심 조건

조건 규칙 해석
공격 조건 금리차↑ + VIX↓ + DXY↓ 상승 우호
방어 조건 금리차↓ + VIX↑ + DXY↑ 리스크 확대
MOVE 구간 ~100 / 100~120 / 120+ 안정 / 주의 / 위험

점수 & 패턴

항목 기준
상승 점수 4~5 우호, 2~3 중립, 0~1 약세
위험 점수 0~1 안정, 2~3 주의, 4+ 위험
패턴 가이드 상승장 초입=적극, 하락장/하락 직전=방어

테마 RSI 기준

종목 구성

  • 우주 50, 양자 50, 원자력 50
  • 총 150종목을 순회 조회
  • 조회 실패 종목은 별도 표시

계산 규칙

  • 지표: RSI14 (Wilder 방식)
  • 장중이면 미완성 캔들 제외
  • 종가는 장마감 기준(previousClose 우선)
